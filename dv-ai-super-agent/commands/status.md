# 프로젝트 상태 확인

현재 프로젝트의 git 상태와 파일 변경사항을 확인합니다.

## 실행할 명령어

```bash
git status
git diff --stat
git log --oneline -5
```

## 출력 형식

### Git 상태
- **브랜치**: [현재 브랜치명]
- **변경된 파일**: [파일 수]개
- **스테이징된 파일**: [파일 수]개
- **커밋되지 않은 변경**: 있음/없음

### 최근 커밋
1. [커밋 해시] [커밋 메시지]
2. ...

### 변경 파일 목록
- [파일 경로]: [추가/수정/삭제]
