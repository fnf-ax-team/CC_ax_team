# 인플루언서 이미지 생성 스킬

> 얼굴 합성으로 인플루언서/셀럽 스타일 이미지 생성 (성별 무관)

## 빠른 시작

```bash
# 1. 패키지 설치
pip install google-genai pillow

# 2. API 키 설정 (.env)
echo "GEMINI_API_KEY=your_key" > .env

# 3. Claude에게 말하기
"인플 이미지 만들어줘"
```

## 트리거 키워드

- `인플`, `인플루언서`, `셀럽`, `셀카 만들어`, `예쁜 사진`

## 사용법

1. **"인플 이미지 만들어줘"** 라고 말하면 스킬 실행
2. 얼굴 사진 2~3장 첨부 (정면 얼빡 추천)
3. 옵션 선택:
   - 스타일: 끼부리는 셀카 / 자연스러운 셀카 / 남이 찍어준 / 거울 셀카
   - 거리: 얼빡 / 상체 / 전신
   - 장소: 카페 / 침대 / 헬스장 / 차 안 / 기타
   - 수량: 1~10장
4. 이미지 생성!

## 핵심 포인트

### 한국어 프롬프트가 더 자연스러움

```
❌ 영어 프롬프트 엔지니어링
"Professional Instagram influencer photo, iPhone 15 Pro Max..."
→ 인위적, 플라스틱 느낌

✅ 한국어로 친구한테 말하듯이
"이 얼굴로 카페에서 턱 괴고 청순하게"
→ 자연스럽고 예쁨
```

### 기술 스펙

| 항목 | 값 |
|------|-----|
| 모델 | `gemini-3-pro-image-preview` |
| Temperature | `0.5` |
| Aspect Ratio | `9:16` (세로) |
| 해상도 | `2K` (2048px) |
