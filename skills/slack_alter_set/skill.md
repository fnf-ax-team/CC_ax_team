## Slack Alter 봇 기능 안내

> 아래는 봇에 내장된 기능 목록입니다. skill.md를 수정해도 이 기능은 변경되지 않습니다.
> 코드를 수정해야 기능이 변경됩니다.

### 응답 방식
| 방식 | 설명 |
|------|------|
| @멘션 | 채널에서 봇을 @멘션하면 응답 |
| DM | 봇에게 직접 메시지를 보내면 응답 |
| 이름 호출 | 채널에서 등록된 이름 패턴을 호출하면 자동 응답 |
| 스레드 | 스레드 내 대화 맥락을 유지하며 응답 |

### 봇 명령어
| 명령어 | 설명 |
|--------|------|
| `!오늘업무` | 최근 7일 회의 기반 오늘 할 일 추출 |
| `!주간브리핑` | 지난주 + 이번주 회의 종합 브리핑 |
| `!회의록` / `!회의록요약` | 회의록 요약 (기본: 지난주) |
| `!회의록 이번주` | 이번주 회의록 요약 |
| `!이번주스케줄` / `!이번주일정` | 이번주 일정 정리 |
| `!날씨` / `!현재날씨` | 현재 날씨 조회 (기본: 설정 지역) |
| `!날씨 {지역}` | 특정 지역 현재 날씨 (예: `!날씨 부산`) |
| `!오늘날씨` / `!날씨예보` | 오늘 날씨 예보 (3시간 간격) |
| `!내일날씨` | 내일까지 날씨 예보 |
| `!지식그래프` / `!kg` | DCS AI 지식그래프 사용 가이드 |
| `!명령어` / `!도움말` | 전체 명령어 안내 |
| `!노션도움말` | Notion 관련 도움말 |
| `!reload` | skill.md 핫 리로드 (관리자 전용) |

### 연동 기능
| 기능 | 설명 |
|------|------|
| **Notion 회의록** | 업무/회의/일정 키워드 감지 시 자동으로 Notion 회의록 컨텍스트 포함 |
| **기상청 날씨** | 날씨 관련 자연어("날씨 어때?", "비 와?", "우산 챙겨야 해?") 감지 시 실시간 기상청 데이터를 자동 첨부하여 AI가 답변 |
| **Outlook 이메일 요약** | Power Automate로 전달된 이메일을 Claude AI로 요약하여 DM 알림 |
| **다중 Notion DB** | `notion_databases.json`으로 여러 Notion 데이터베이스 동시 연동 |

### 이름 호출 패턴 설정

봇이 반응할 이름 패턴은 `handlers/message.py`의 `NAME_PATTERNS` 리스트에서 설정합니다.

```python
# handlers/message.py 예시
NAME_PATTERNS = [
    "홍길동 과장",   # 풀네임 + 직급
    "길동님",         # 이름 + 님
    "길동씨",         # 이름 + 씨
    "홍과장",         # 성 + 직급
    "홍길동",         # 풀네임
]
```

## 커스터마이징 가이드

### 이 파일(skill.md)에서 설정하는 것
- 봇의 페르소나 (이름, 말투, 톤)
- 담당 업무 지식 및 대응 방법
- FAQ 및 에스컬레이션 규칙
- 프로젝트 컨텍스트
- 스킬셋 및 역량 수준

### 코드에서 설정하는 것
- 이름 호출 패턴: `handlers/message.py` → `NAME_PATTERNS`
- 날씨 기본 지역: `.env` → `WEATHER_DEFAULT_LOCATION` (미설정 시 강남구)
- Outlook 전달 채널: `.env` → `OUTLOOK_FORWARD_CHANNEL`
- Notion DB 연동: `notion_databases.json`
- 관리자 권한: `.env` → `MY_REAL_USER_ID`

### 환경 변수 (.env)
```bash
# 필수
SLACK_BOT_TOKEN=xoxb-...          # Slack Bot Token
SLACK_APP_TOKEN=xapp-...          # Slack App Token (Socket Mode)
ANTHROPIC_API_KEY=sk-ant-...      # Claude API Key
BOT_USER_ID=U...                  # 봇의 Slack User ID
MY_REAL_USER_ID=U...              # 관리자의 Slack User ID (!reload 권한)

# Notion 연동 (선택)
NOTION_API_KEY=secret_...         # Notion Integration Token
NOTION_DATABASE_ID=...            # 기본 Notion DB ID (notion_databases.json 미사용 시)
NOTION_DATE_PROPERTY=날짜          # Notion DB의 날짜 속성명
NOTION_USER_NAME=홍길동            # 회의록 분석 시 사용할 이름

# 날씨 연동 (선택)
KMA_AUTH_KEY=...                  # 기상청 API 허브 인증키 (https://apihub.kma.go.kr)
WEATHER_DEFAULT_LOCATION=강남구    # 기본 날씨 조회 지역 (미설정 시 강남구)

# Outlook 연동 (선택)
OUTLOOK_FORWARD_CHANNEL=C...      # Outlook 이메일 전달 채널 ID
MY_SLACK_DM_CHANNEL=D...          # 요약 알림을 받을 DM 채널 ID

# 기타
SKILL_FILE_PATH=skill.md          # skill 파일 경로 (기본값: skill.md)
```
