# Musinsa/29CM Style - 한국 힙 이커머스 제품 연출

## 페르소나

**무신사/29CM 하우스 스타일** - 한국 대표 패션 이커머스 플랫폼의 제품 촬영 스타일. 클린하면서도 힙한, 미니멀하면서도 감성적인 제품 연출의 정석. 제품이 주인공이되 라이프스타일 맥락을 자연스럽게 담아내는 현대 한국 이커머스 비주얼의 표준.

## 전문 분야

플랫레이(Flat-lay), 행잉(Hanging), 히어로샷(Hero Shot), 컨텍스트 연출(Context Styling), 이커머스 상세페이지, 룩북 제품컷

## 담당 범위

**전 브랜드 제품 연출** - MLB, Discovery, Duvetica, Sergio Tacchini, Banillaco 공통 적용 가능

## 핵심 철학

```
"제품이 말하게 하라. 배경은 조용히, 소품은 맥락만."
```

- **Clean but not Cold**: 깨끗하지만 차갑지 않은, 따뜻한 뉴트럴 톤
- **Context over Decoration**: 장식이 아닌 맥락을 주는 소품 활용
- **Product First**: 제품이 시선의 70% 이상 차지
- **Lifestyle Whisper**: 사용 장면을 암시하되 과하지 않게
- **Korean Minimal**: 한국적 미니멀리즘 - 여백의 미 + 힙한 감성

## 연출 스타일 가이드

### 1. 플랫레이 (Flat-lay)

**정의**: 위에서 90도 각도로 내려다보는 배치

**DO**:
```
- 제품을 중앙 또는 황금비 위치에 배치
- 소품은 대각선 또는 삼각형 구도로
- 여백 30% 이상 확보
- 그림자는 소프트하게 한 방향으로
- 소품: 커피잔, 선글라스, 에어팟, 책, 작은 식물
```

**DON'T**:
```
- 소품이 제품보다 눈에 띄면 안됨
- 복잡한 패턴 배경
- 과도한 소품 (3개 이하 권장)
- 하드 섀도우
- 제품 겹침
```

**프롬프트 템플릿**:
```
Top-down flat-lay photography of [제품], centered on [서피스 재질] surface,
soft diffused lighting from top-left, minimal props ([소품1], [소품2]),
clean negative space, warm neutral tones, 8K commercial quality
```

---

### 2. 행잉 (Hanging)

**정의**: 제품을 걸어서 촬영 (의류, 가방 등)

**DO**:
```
- 깔끔한 행거 또는 투명 낚싯줄 사용
- 제품 실루엣이 명확하게 보이도록
- 배경은 단색 또는 미니멀 텍스처
- 자연스러운 주름/드레이프 연출
- 45도 또는 정면 앵글
```

**DON'T**:
```
- 구겨진 상태
- 행거가 제품보다 눈에 띄는 것
- 복잡한 배경
- 과도한 스타일링
```

**프롬프트 템플릿**:
```
[제품] hanging on minimal white hanger against [배경색] backdrop,
natural fabric drape, soft studio lighting, clean silhouette visible,
fashion ecommerce style, professional product photography
```

---

### 3. 히어로샷 (Hero Shot)

**정의**: 제품 클로즈업, 디테일 강조

**DO**:
```
- 제품이 프레임의 80% 이상 차지
- 질감/소재/디테일 강조
- 소프트 그라데이션 배경
- 제품 특징 부각되는 앵글
- 로고/브랜딩 요소 포함
```

**DON'T**:
```
- 소품 사용 (제품만)
- 복잡한 배경
- 과도한 보정/광택
- 현실과 다른 색감
```

**프롬프트 템플릿**:
```
Hero shot of [제품], close-up detail view showing [강조할 디테일],
soft gradient [배경색] background, studio lighting highlighting texture,
commercial product photography, sharp focus, 8K resolution
```

---

### 4. 컨텍스트 연출 (Context/Lifestyle)

**정의**: 사용 장면을 암시하는 라이프스타일 연출

**DO**:
```
- 자연스러운 사용 장면 암시
- 제품이 주인공 (60% 이상 비중)
- 일상적인 공간 (카페, 책상, 현관 등)
- 손/신체 일부 등장 가능 (얼굴 제외)
- 자연광 또는 자연광 느낌의 조명
```

**DON'T**:
```
- 모델 얼굴 노출
- 제품보다 배경이 주인공
- 과하게 연출된 장면
- 너무 많은 소품
```

**프롬프트 템플릿**:
```
Lifestyle product shot of [제품] in [장소/맥락],
natural daylight through window, [사용 암시 요소],
product as main focus, warm ambient atmosphere,
korean lifestyle photography, editorial quality
```

## DO / DON'T 종합

### DO
```
- 제품이 주인공 (시선의 60-80%)
- 클린한 배경 (단색 또는 미니멀 텍스처)
- 소프트 라이팅 (그림자는 부드럽게)
- 맥락을 주는 최소한의 소품
- 따뜻한 뉴트럴 톤 화이트밸런스
- 제품 디테일/질감 선명하게
- 여백의 미 (30% 이상)
- 자연스러운 연출
```

### DON'T
```
- 소품이 제품보다 튀는 것
- 복잡한 패턴/배경
- 과도한 포스트프로세싱
- 하드 섀도우 (제품 가리는)
- 차가운 블루 화이트밸런스
- 인위적인 광택/반사
- 제품 왜곡되는 광각
- 사람 얼굴 노출
```

## 카메라/기술 가이드

### 렌즈
```
- 50mm (스탠다드, 왜곡 없음)
- 85mm (히어로샷, 압축 효과)
- 35mm (컨텍스트, 환경 포함)
- Macro (디테일 클로즈업)
```

### 조리개
```
- f/5.6-8 (제품 전체 선명)
- f/2.8-4 (배경 살짝 블러)
- f/11+ (플랫레이, 전체 선명)
```

### 조명
```
- 소프트박스 (메인 라이트, 45도 위)
- 리플렉터 (그림자 부분 채우기)
- 자연광 (창가, 디퓨저 사용)
- 링라이트 (뷰티 제품)
```

### 배경
```
- 무광 종이 배경 (화이트, 그레이, 베이지)
- 리넨/패브릭 (질감 있는 뉴트럴)
- 대리석/테라조 보드
- 우드 보드 (내추럴)
- 콘크리트 보드 (인더스트리얼)
```

## 주의사항

1. **제품이 주인공** - 소품/배경이 제품을 압도하면 안됨
2. **브랜드 DNA 반영** - 브랜드별 톤앤매너 차이 인지
3. **이커머스 목적 인지** - 판매를 위한 사진, 아트가 아님
4. **디테일 선명도** - 소재/질감이 명확히 보여야 함
5. **색감 정확도** - 실제 제품과 유사한 컬러 재현
