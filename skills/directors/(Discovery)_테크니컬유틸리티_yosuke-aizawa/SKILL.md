## Discovery 촬영 가이드

### 모델 방향

- 지적이고 차분한 페이스
- 세련된 현대적 탐험가 이미지
- 자연과 건축물 사이의 조화로운 태도

### 로케이션 참조

```
OK:
- 노출 콘크리트와 숲이 공존하는 곳
- 미니멀한 현대적 산장
- 정제된 암석 지대
- 도시의 직선미가 느껴지는 공원

NO:
- 지저분한 캠핑장
- 과도하게 험한 산악
- 스트릿/도심 골목
```

### 포즈 레퍼런스

- 정적인 시선 처리
- 장비의 기능을 점검하는 듯한 동작
- 바람막이 후드나 지퍼를 활용한 디테일 포즈
- 안정적이고 여유로운 스탠스

### 조명 방향

- 부드러운 측면 자연광
- 차가운 톤의 블루아워 광원
- 소재의 입체감을 살리는 소프트 섀도우
- 은은한 하이라이트로 테크니컬 질감 강조

## 주의사항

1. **아웃도어 ≠ 거칠음** - 지나치게 투박해 보이지 않게
2. **기능미 ≠ 테크웨어** - 난잡한 사이버펑크로 흐르지 않게
3. **프리미엄 정체성** - 도심에서도 이질감 없는 톤 유지
4. **스마트 라이프스타일** - Discovery 특유의 지적인 톤앤매너 고수
