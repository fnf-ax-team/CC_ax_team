# 셀카/인플루언서 스타일 이미지 생성

> 분위기 선택으로 꾸민 느낌 ~ 자연스러운 느낌까지

## 개요

셀카/인플루언서 스타일 이미지를 생성하는 Claude Code 스킬입니다.
Gemini API를 사용하여 참조 얼굴 이미지 기반으로 다양한 스타일의 셀카를 생성합니다.

## 파일 구조

| 파일 | 설명 |
|------|------|
| `SKILL.md` | 스킬 명세서 - 워크플로, 모듈 인터페이스, 검증 기준 |
| `selfie-prompt-cheatsheet.md` | 프롬프트 가이드 - 옵션 매핑, 예시, 금지 조합 |
| `validator.py` | 검증 모듈 - 5개 기준 품질 검사 |

## 핵심 원칙

**"너무 잘 나오면 실패"** - UGC/셀카는 자연스러운 불완전함이 있어야 합니다.

## 5개 검증 기준

| 기준 | 비중 | 설명 |
|------|------|------|
| realism | 35% | 실제 폰카 사진처럼 보이는가 |
| person_preservation | 25% | 참조 얼굴과 같은 사람인가 |
| scenario_fit | 20% | 장소/상황/옷이 어울리는가 |
| skin_condition | 10% | 피부 질감이 자연스러운가 |
| anti_polish_factor | 10% | 너무 완벽하지 않은가 |

## 분위기 옵션

| 분위기 | 설명 |
|--------|------|
| 꾸민 느낌 | 인플루언서 스타일 - 링라이트, 풀메이크업 |
| 자연스러운 | 일상 캐주얼 - 자연광, 민낯/내추럴 |

## Auto-Fail 조건

- 손가락 6개 이상 / 기형적 손가락
- 얼굴 다른 사람
- 누런 톤 (golden/amber/warm cast)
- AI 플라스틱 피부
- 의도하지 않은 텍스트/워터마크
