# API 엔드포인트 확인

백엔드 API 라우트 파일들을 분석하여 사용 가능한 엔드포인트를 정리합니다.

## 분석할 파일

`backend/src/routes/` 디렉토리의 모든 `*.routes.ts` 파일을 읽어주세요.

## 출력 형식

### API 엔드포인트 목록

| Method | Endpoint | 설명 |
|--------|----------|------|
| GET | /api/kpi | KPI 요약 |
| GET | /api/categories | 카테고리별 매출 |
| ... | ... | ... |

### 주요 API 파라미터

각 API의 쿼리 파라미터나 요청 본문 구조를 간단히 설명해주세요.
