## Executive Summary

**"브랜드 DNA를 한 번 정의하면, AI가 알아서 지킵니다."**

- 브랜드별 치트시트에 톤 & 무드 규칙 정의
- AI가 이미지 생성 시 자동으로 규칙 준수
- 5단계 자동 검수로 브랜드 톤 위반 차단
- 위반 시 자동 재생성 → 품질 보장된 결과물만 전달

---

## 1. 핵심 질문과 답변

### Q: 브랜드별 톤을 어떻게 맞추나요?

**A: 3단계 자동화 시스템으로 관리합니다.**

| 단계 | 내용 | 자동화 |
|------|------|--------|
| 1단계 | 브랜드 치트시트에 DNA 정의 | 1회 작성 |
| 2단계 | AI가 치트시트 규칙 자동 준수 | 완전 자동 |
| 3단계 | 5-Gate 검수로 톤 위반 자동 탈락 | 완전 자동 |

## 3. 브랜드 치트시트란?

### 정의
브랜드의 비주얼 DNA를 JSON 스키마 형태로 정의한 문서

### 구성 요소

| 항목 | 설명 | 예시 (MLB) |
|------|------|-----------|
| **브랜드 DNA** | DO & DON'T 명확 정의 | 쿨톤만, 골든아워 금지 |
| **금지 조합** | 절대 사용 금지 조합 | 85mm + 전신샷 |
| **프롬프트 옵션** | 허용된 값 목록 | 표정: cool, natural, dreamy |
| **기본값** | 미지정 시 자동 적용 | 헤어: 블랙, 스트레이트 |
| **호환 규칙** | 조합 가능 여부 | 차량 배경 ↔ 기대는 포즈 |

### 예시: MLB 치트시트 핵심

```
DO (허용)                      DON'T (금지)
─────────────────────────────────────────────
쿨톤, 뉴트럴 색감              골든아워, 웜톤, 앰버
큰 아몬드눈                    찡그린 눈, 작은 눈
쿨하고 무심한 표정             밝은 미소, 치아 노출
콘크리트, 메탈, 산업적 배경     자연 풍경, 따뜻한 우드톤
미드리프 노출, 오버사이즈       클래식, 포멀 스타일
```

## 4. 자동 검수 시스템

### 5-Gate 순차 검증

| Gate | 체크 항목 | 탈락 조건 | 브랜드 톤 관련 |
|------|----------|----------|---------------|
| 1 | 착장 | 색상/로고/디테일 불일치 | - |
| 2 | 얼굴 | 다른 사람 | - |
| 3 | 포즈 | 관절/손가락 어색 | - |
| **4** | **브랜드 톤** | **치트시트 금지 요소 위반** | **핵심** |
| 5 | 품질 마감 | 해상도/아티팩트 | - |

### Gate 4 상세: 브랜드 톤 검증

```
검증 항목:
├── 색감: 쿨톤 유지? 웜톤 침범?
├── 표정: 브랜드 DNA 준수?
├── 배경: 금지된 배경 타입?
├── 스타일링: 금지 조합 사용?
└── 전체 무드: 브랜드 아이덴티티 일치?

위반 시:
└── 즉시 탈락 → 원인 진단 → 프롬프트 수정 → 재생성
```

## 5. 새 브랜드 추가 프로세스

### 소요 시간: 1일

```
Step 1: 기존 촬영본 수집 (30분)
        └── 브랜드 대표 화보 10-20장

Step 2: VLM 분석 (1시간)
        └── AI가 패턴 자동 추출
            - 색감 톤
            - 표정 유형
            - 포즈 스타일
            - 배경 타입

Step 3: 치트시트 작성 (2-3시간)
        └── 분석 결과 기반 DNA 정의
            - DO & DON'T
            - 금지 조합
            - 기본값

Step 4: 테스트 생성 (1시간)
        └── 5-10장 생성하여 톤 검증

Step 5: 즉시 운영 적용
        └── 치트시트 배포 → AI 자동 로드
```
