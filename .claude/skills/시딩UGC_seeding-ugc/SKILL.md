# 시딩 UGC

> 광고 아닌 척, 진짜 사용자 콘텐츠처럼 보이는 이미지 생성

---

## 절대 규칙 (CRITICAL)

1. **UGC 진정성** - 광고티 나면 실패. 실제 사용자가 올린 것처럼 보여야 함
2. **"너무 완벽하면 실패"** - 에디토리얼/화보 퀄리티는 오히려 감점
3. **상품 자연스러운 노출** - 너무 강조하면 광고, 안 보이면 마케팅 효과 없음

---

## 실행 파이프라인

```
1. 브랜드 라우팅   → 브랜드 DNA + 시나리오 판정
2. 프롬프트 조립   → 시나리오별 프롬프트 생성
3. 이미지 생성     → 병렬 생성 (9:16, 2K)
4. UGC 검증       → 리얼리즘 + 안티폴리시 체크
5. 재시도         → 실패 시 프롬프트 강화 + 온도 증가
6. 시딩 가이드    → 캡션/해시태그 자동 생성
```

## 대화형 질문

| 순서 | 질문 | 필수 | 기본값 |
|------|------|------|--------|
| 1 | 모델 이미지 | ✅ | - |
| 2 | 브랜드 | ✅ | - |
| 3 | 시나리오 | ✅ | 자동 판정 |
| 4 | 비율 | ✅ | 9:16 |
| 5 | 수량 | ✅ | 3장 |
| 6 | 화질 | ✅ | 2K |

### 시나리오 옵션

| 카테고리 | 시나리오 |
|----------|----------|
| pain_point | 두통/햇빛, 번들거림, 트러블, 건조함, 다크서클, 바람 |
| before_after | 스킨케어 전, 스킨케어 후 |
| daily_routine | 아침 루틴, 운동 후 |

## 검증 기준

| 항목 | 비중 | 설명 |
|------|------|------|
| realism | 30% | 실제 폰카처럼 보이는가 |
| person_preservation | 25% | 같은 사람인가 |
| scenario_fit | 15% | 상황이 자연스러운가 |
| product_visibility | 15% | 상품이 자연스럽게 노출되는가 (너무 강조 X) |
| skin_condition | 8% | 피부가 자연스러운가 (플라스틱 금지) |
| anti_polish_factor | 7% | 너무 완벽하지 않은가 |

**Auto-Fail**:
- AI 플라스틱 피부
- 스튜디오 배경
- 전문 모델 포즈
- 상품 로고 클로즈업
- 명백한 광고 문구
- 너무 완벽함

**등급**: S/A(90+) 즉시 사용, B(85+) 소폭 보정, C/F(85-) 재생성
