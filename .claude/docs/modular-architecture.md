## 요약: 분석기 1번 잘 만들면

```
 ════════════════════════════════════════════════════════════════

  ┌─────────────────────┐     ┌─────────────────────┐
  │ DB 축적 (용도 1)     │     │ 실시간 분석 (용도 2)  │
  │                     │     │                     │
  │ 새 브랜드 화보       │     │ 사용자가 넣는        │
  │ → 배치 돌리면        │     │ 아무 이미지          │
  │ → 프리셋 자동 생성   │     │ → 같은 정밀도로 분석  │
  │ → 코드 수정 없음     │     │ → 새 워크플로에도     │
  │                     │     │   그대로 재사용       │
  │ MLB → Discovery     │     │                     │
  │ → Duvetica → ...    │     │ BC, INF, BG, OS     │
  │                     │     │ + 포즈따라하기        │
  │ DB만 계속 커짐       │     │ + 포즈변경 + ...     │
  └─────────────────────┘     └─────────────────────┘

  분석 품질 개선 시 → 분석기 1곳만 수정 → 전체 반영
  새 워크플로 추가 시 → 기존 분석기 조합 + 설정만
  새 브랜드 추가 시 → 화보 넣고 배치만 돌리면 끝

 ════════════════════════════════════════════════════════════════
```
