# 주문 도메인

## 개요
고객의 상품 주문을 처리하는 핵심 도메인

## 주요 개념

### 주문 상태 (Order Status)
| 상태 | 코드 | 설명 |
|------|------|------|
| 주문 접수 | PENDING | 결제 전 상태 |
| 결제 완료 | PAID | 결제 확인됨 |
| 배송 준비 | PREPARING | 상품 준비 중 |
| 배송 중 | SHIPPING | 배송 시작 |
| 배송 완료 | DELIVERED | 고객 수령 완료 |
| 취소 | CANCELLED | 주문 취소 |
| 환불 | REFUNDED | 환불 처리 완료 |

### 비즈니스 규칙
- 결제 완료 후 24시간 내 무료 취소 가능
- 배송 시작 후 취소 시 배송비 차감
- 부분 취소 가능 (개별 상품 단위)

## 관련 테이블
- `orders` - 주문 헤더
- `order_items` - 주문 상세
- `order_status_history` - 상태 변경 이력

## 담당자
- 백엔드: 개발팀 김개발
- 데이터: 데이터팀 이분석
