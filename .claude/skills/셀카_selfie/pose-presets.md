# 포즈 프리셋 v3.0

> VLM 분석 기반 고품질 포즈 프리셋 (75종)
>
> **배경/장소/의상 정보 제외** - 순수 포즈만 포함

---

## v3.0 변경사항

| 항목 | v2.0 | v3.0 |
|------|------|------|
| 한줄요약 | 없음 | 포즈 전체 인상을 직관적으로 설명 |
| 태그 | 없음 | 검색/분류용 키워드 3~7개 |
| 가시성 | 안 보이는 부위를 추측 | "보이지않음" 명시 |
| 스키마 소스 | 이 문서에 별도 관리 | `db/presets/common/pose_presets.json` (Single Source of Truth) |

### 핵심 규칙

- **보이지 않는 부위 = "보이지않음"** (추측/지어내기 금지)
- **한줄요약은 직관적** (관절 각도 나열 X, 시각적 인상 O)
- **코어 분석기**: `core/ai_influencer/pose_analyzer.py`
- **재분석 스크립트**: `scripts/reanalyze_pose_presets_v3.py`
