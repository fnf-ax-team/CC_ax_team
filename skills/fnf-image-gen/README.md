# FNF 이미지 생성 스킬

> FNF Studio AI 이미지 생성 스킬 모음

## 개요

FNF AI Studio에서 사용하는 이미지 생성 스킬들입니다.
Gemini API 기반으로 다양한 워크플로를 실행합니다.

| 스킬 | 설명 | 키워드 |

| [셀카](./셀카_selfie/) | 셀카/인플루언서 스타일 이미지 | `셀카`, `셀피`, `인플` |

## 공통 규칙

### 필수 모델

| 용도 | 모델 |
|------|------|
| 이미지 생성 | `gemini-3-pro-image-preview` |
| VLM 분석 | `gemini-3-flash-preview` |

### 금지 모델

```
gemini-2.0-flash-exp
gemini-2.0-flash
gemini-2.5-flash
```

### 4단계 워크플로 구조

모든 이미지 생성 워크플로는 다음 구조를 따릅니다:

```
1. 분석 (VLM)     - gemini-3-flash-preview로 참조 이미지 분석
2. 생성 (Image)   - gemini-3-pro-image-preview로 이미지 생성
3. 검수 (VLM)     - 워크플로별 검수 기준으로 품질 판정
4. 재생성 (Loop)  - 탈락 시 원인 진단 → 프롬프트 수정 → 재생성 (최대 2회)
```

| 9:16 | 스토리/릴스 |
| 16:9 | 유튜브/가로 영상 |

## 스킬 구조

각 스킬 폴더에는 다음 파일들이 포함됩니다:

| 파일 | 설명 |
|------|------|
| `SKILL.md` | 스킬 명세서 (워크플로, 모듈 인터페이스) |
| `*-prompt-cheatsheet.md` | 프롬프트 가이드 (옵션 매핑, 예시) |
| `validator.py` | 검증 모듈 (품질 검사) |
| `README.md` | 스킬 개요 |

## Auto-Fail 공통 조건

다음 조건에 해당하면 모든 워크플로에서 자동 탈락:

- 손가락 6개 이상 / 기형적 손가락
- 얼굴 다른 사람 (참조와 불일치)
- 누런 톤 (golden/amber/warm cast)
- AI 플라스틱 피부
- 의도하지 않은 텍스트/워터마크
